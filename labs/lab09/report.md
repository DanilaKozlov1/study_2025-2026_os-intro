# Цель работы

Освоение основных возможностей командной оболочки Midnight Commander. Приобретение навыков практической работы по просмотру каталогов и файлов; манипуляций с ними.

# Задание

1. Изучить структуру и меню Midnight Commander.
2. Выполнить основные операции с файлами и каталогами в mc.
3. Освоить встроенный редактор mcedit.

# Теоретическое введение

Midnight Commander (mc) — псевдографическая командная оболочка для UNIX/Linux систем. Рабочее пространство mc имеет две панели, отображающие списки файлов каталогов. Управление осуществляется с помощью функциональных клавиш F1–F10 и комбинаций клавиш.

| Клавиша | Действие |
|---------|----------|
| F1 | Вызов справки |
| F3 | Просмотр файла |
| F4 | Редактирование файла |
| F5 | Копирование |
| F6 | Перемещение/переименование |
| F7 | Создание каталога |
| F8 | Удаление |
| F9 | Вызов меню |
| F10 | Выход |

# Выполнение лабораторной работы

## Задание по mc

Установил mc командой `sudo dnf install mc` и запустил его из терминала командой `mc`.

Переключился между панелями клавишей Tab. Поменял панели местами комбинацией Ctrl-u. Скрыл панели комбинацией Ctrl-o и вернул их повторным нажатием Ctrl-o.

Выделил несколько файлов клавишей Ctrl-t и скопировал их клавишей F5 в каталог, отображаемый на второй панели. Переместил файл с помощью F6.

Через меню «Левая панель» (F9) переключил режим отображения на «Быстрый просмотр» — на панели отобразилось содержимое выделенного файла. Переключил режим на «Информация» — отобразились сведения о файле (права, размер, владелец). Переключил на «Дерево» — отобразилась структура каталогов.

Изменил формат списка на «Расширенный» — стали видны права доступа, владелец, группа. Переключил на «Укороченный» — отобразились только имена файлов в несколько столбцов. Вернул «Стандартный» формат.

Изменил порядок сортировки на «Размер», затем вернул сортировку по имени.

Просмотрел текстовый файл с помощью F3 (без возможности редактирования). Открыл файл на редактирование с помощью F4, внёс изменения, но не сохранил их при выходе (F10 → «Нет»).

Создал каталог test_dir с помощью F7. Скопировал файлы в test_dir с помощью F5.

Выполнил поиск файла через меню «Команда» → «Поиск файла»: искал файлы `*.c` с содержимым `main`, начиная с каталога `/usr`.

Открыл историю командной строки через меню «Команда» → «История командной строки», выбрал и повторил одну из предыдущих команд.

Перешёл в домашний каталог командой `cd ~`.

Открыл и проанализировал файл расширений через меню «Команда» → «Редактировать файл расширений» — в нём описаны действия при открытии файлов с различными расширениями. Открыл файл меню через «Редактировать файл меню» — в нём описаны пункты пользовательского меню, вызываемого по F2.

Открыл меню «Настройки» → «Конфигурация» и ознакомился с параметрами. Через «Внешний вид» временно отключил строку подсказок. Через «Настройки панелей» включил показ скрытых файлов — в панели появились файлы, начинающиеся с точки (.bashrc, .bash_profile и т.д.). Ознакомился с пунктами «Подтверждение», «Биты символов», «Распознание клавиш».

## Задание по встроенному редактору mc

Создал файл text.txt командой `touch text.txt` и открыл его в редакторе клавишей F4. Набрал несколько строк текста.

Удалил строку комбинацией Ctrl-y.

Выделил фрагмент текста (F3 — начало выделения, стрелками выделил текст, F3 — конец выделения), скопировал его на новую строку клавишей F5.

Аналогично выделил другой фрагмент (F3), переместил его на новую строку клавишей F6.

Сохранил файл клавишей F2.

Отменил последнее действие комбинацией Ctrl-u.

Перешёл в конец файла
комбинацией Ctrl-End и дописал текст. Перешёл в начало файла комбинацией Ctrl-Home и дописал текст.

Сохранил файл (F2) и вышел из редактора (F10).

Создал файл hello.c, открыл его в редакторе (F4) и написал программу на языке Си. Подсветка синтаксиса была включена — ключевые слова отображались разными цветами.

Через меню редактора (F9 → «Настройки») выключил подсветку синтаксиса, затем включил обратно.
