# Теоретическое введение

Редактор vi (Visual display editor) — текстовый редактор, установленный по умолчанию практически во всех дистрибутивах Linux. Редактор vi имеет три режима работы: командный режим (ввод команд редактирования и навигации), режим вставки (ввод содержания файла) и режим последней строки (запись изменений и выход).

| Команда | Действие |
|---------|----------|
| i | Вставить текст перед курсором |
| a | Вставить текст после курсора |
| o | Вставить строку под курсором |
| dd | Удалить строку |
| dw | Удалить слово |
| u | Отменить последнее изменение |
| :wq | Сохранить и выйти |
| :q! | Выйти без сохранения |

# Контрольные вопросы

1. Редактор vi имеет три режима работы. Командный режим — режим по умолчанию, в котором нажатия клавиш интерпретируются как команды редактирования и навигации. Режим вставки — режим ввода текста, активируется клавишами i, a, o и другими; введённые символы вставляются в текст. Режим последней строки — режим ввода команд в нижней строке экрана, активируется нажатием `:` из командного режима; используется для сохранения, выхода, поиска и замены.

2. Чтобы выйти из редактора без сохранения изменений, нужно нажать `Esc` для перехода в командный режим, затем набрать `:q!` и нажать Enter. Символ `!` принудительно закрывает редактор, игнорируя несохранённые изменения.

3. Команды позиционирования: `0` (ноль) — переход в начало строки; `$` — переход в конец строки; `G` — переход в конец файла; `nG` — переход на строку с номером n.

4. Для редактора vi словом является последовательность символов, разделённых пробелами, табуляциями или символами новой строки. При использовании команд с маленькими буквами (w, b) разделителями также считаются знаки пунктуации. При использовании команд с большими буквами (W, B) разделителями считаются только пробел, табуляция и возврат каретки.

5. Чтобы перейти в начало файла, нужно в командном режиме нажать `gg` или `1G`. Чтобы перейти в конец файла, нужно нажать `G`.

6. Основные группы команд редактирования: вставка текста (i, a, I, A — вставка в различные позиции относительно курсора); вставка строк (o, O — создание новой строки под или над курсором); удаление текста (x — символ, dw — слово, dd — строка, d$ — до конца строки); копирование (Y — строка, yw — слово); вставка из буфера (p — после курсора, P — перед курсором); замена (cw — заменить слово, r — заменить символ, R — режим замены); отмена и повтор (u — отмена, `.` — повтор).

7. Чтобы заполнить строку символами `$`, нужно перейти в режим вставки клавишей `i` и вручную набрать нужное количество символов `$`. Альтернативный способ: в командном режиме набрать, например, `80i$` и нажать `Esc` — это вставит символ `$` 80 раз.

8. Чтобы отменить некорректное действие, нужно нажать `Esc` для перехода в командный режим и нажать `u`. Команда `u` отменяет последнее изменение. Для отмены всех изменений в текущей строке можно нажать `U`.

9. Основные группы команд режима последней строки: запись и выход (`:w` — сохранить, `:q` — выйти, `:wq` — сохранить и выйти, `:q!` — выйти без сохранения); копирование и перемещение строк (`:n,m d` — удалить строки, `:i,j m k` — переместить строки, `:i,j t k` — копировать строки); запись диапазона в файл (`:i,j w имя-файла`); отмена всех изменений (`:e!`).

10. Чтобы определить позицию конца строки, не перемещая курсора, можно использовать команду `:set list` в режиме последней строки. Эта команда отобразит невидимые символы, и конец каждой строки будет отмечен символом `$`.

11. Для просмотра полного списка опций редактора vi нужно в режиме последней строки набрать `:set all`. Будет выведен список всех доступных опций с их текущими значениями. Основные опции: `nu` — нумерация строк, `list` — отображение невидимых символов, `ic` — игнорирование регистра при поиске. Для включения опции используется `:set имя`, для отключения — `:set nоимя`.

12. Определить текущий режим работы редактора vi можно по нижней строке экрана. В режиме вставки внизу отображается надпись `-- INSERT --`. В режиме замены — `-- REPLACE --`. В режиме последней строки внизу отображается двоеточие `:` с курсором. Если внизу нет никаких надписей — редактор находится в командном режиме.

13. Граф взаимосвязи режимов работы редактора vi: из командного режима можно перейти в режим вставки (клавиши i, a, o, O, I, A) и в режим последней строки (клавиша `:`). Из режима вставки можно вернуться только в командный режим (клавиша `Esc`). Из режима последней строки можно вернуться в командный режим (клавиша `Esc` или выполнение команды). Таким образом, командный режим является центральным — через него осуществляется переход между двумя другими режимами.
