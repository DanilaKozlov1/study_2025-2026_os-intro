# Цель работы

Приобретение практических навыков взаимодействия пользователя с системой посредством командной строки.

# Задание

Изучить основные команды командной строки Linux: навигация по файловой системе, просмотр содержимого каталогов, создание и удаление каталогов, работа с историей команд.

# Теоретическое введение

В Linux взаимодействие с системой осуществляется через командные интерпретаторы (shell). Команда имеет формат: `<имя_команды> <аргументы>`. Основные команды: `pwd`, `cd`, `ls`, `mkdir`, `rmdir`, `rm`, `man`, `history`.

| Имя каталога | Описание каталога                                                                                                          |
|--------------|----------------------------------------------------------------------------------------------------------------------------|
| `/`          | Корневая директория, содержащая всю файловую                                                                               |
| `/bin`       | Основные системные утилиты, необходимые как в однопользовательском режиме, так и при обычной работе всем пользователям     |
| `/etc`       | Общесистемные конфигурационные файлы и файлы конфигурации установленных программ                                           |
| `/home`      | Содержит домашние директории пользователей, которые содержат персональные настройки и данные пользователя                  |
| `/media`     | Точки монтирования для сменных носителей                                                                                   |
| `/root`      | Домашняя директория пользователя `root`                                                                                    |
| `/tmp`       | Временные файлы                                                                                                            |
| `/usr`       | Вторичная иерархия для данных пользователя                                                                                 |

# Ответы на контрольные вопросы

1. Командная строка — текстовый интерфейс для ввода команд управления ОС.
2. `pwd` — выводит абсолютный путь текущего каталога. Пример: `pwd` → `/home/student`.
3. `ls -F` — показывает имена и типы файлов (`/` — каталог, `*` — исполняемый, `@` — ссылка).
4. `ls -a` — отображает скрытые файлы (начинающиеся с точки).
5. Файл удаляется через `rm файл`, каталог — через `rmdir каталог` или `rm -r каталог`. Командой `rm -r` можно удалить и то, и другое.
6. История команд выводится командой
`history`.
7. Модификация команды из истории: `!3:s/-a/-la` — заменит опцию в команде №3 и выполнит её.
8. Несколько команд в одной строке: `cd; ls` или `pwd; ls -l`.
9. Символ экранирования `\` заставляет воспринимать спецсимволы буквально. Пример: `ls \*.txt`.
10. `ls -l` выводит: тип и права доступа, число ссылок, владельца, размер, дату изменения, имя файла.
11. Относительный путь задаётся от текущего каталога, абсолютный — от корня. Пример: `cd newdir` (относительный) и `cd /home/student/newdir` (абсолютный).
12. Справку по команде можно получить через `man <команда>`.
13. Автодополнение команд — клавиша **Tab**.
